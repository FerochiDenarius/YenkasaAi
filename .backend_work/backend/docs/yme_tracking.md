# YME Tracking

Every successful AI interaction creates a YME event with:

- `event_type`
- `user_id`
- `session_id`
- `timestamp`
- `source`
- `metadata`

Current AI metadata includes:

- feature name
- inferred topics
- inferred coding languages
- response latency
- estimated token usage

This is the base for:

- long-term user memory
- coding-interest clustering
- future personalization
- ecosystem intelligence graphing
